# Cocos v2 产品设计（GDD）

> **Status:** Target  
> **Applies to:** Cocos 独立 App v2  
> **Last verified:** 2026-07-15

系统分册只细化机制，不得改变本文件的产品规则。总体机制细节见设计规格 `docs/superpowers/specs/2026-07-15-language-astronauts-cocos-redesign-design.md`。

## 1. 一句话定义

《时空语航员》是一款把教材同步英语练习转化为**电影化指令 RPG** 的离线儿童学习游戏。

## 2. 设计目标（优先级从高到低）

1. 到期知识点的首次回忆正确率持续提升；
2. 孩子愿意主动完成短时、持续的复习；
3. 听、说、读、写题型都获得足够清晰的交互空间；
4. 战斗、武器和宠物拥有可感知的机制差异与演出差异；
5. 家长能理解孩子学了什么、掌握到什么程度；
6. 个人开发者能够持续生产和维护内容。

## 3. 非目标

- 不做实时 PvP、排行榜、公会或社交聊天；
- 不做抽卡、体力付费、广告、内购、连续签到惩罚；
- 不以总游戏时长为成功指标；
- 不以手速、闪避或复杂操作决定学习成败；
- 不让装备、宠物或等级修改正确答案、掌握判定或复习时间；
- 首发不建设服务器和云账号（**完全离线**）。

## 4. 分龄与无障碍

### 1–3 年级

- 无强制答题倒计时；默认 2×2 大按钮，最小触控高度 **64 dp**；
- 长句选项改为 1×4 纵向布局；拼写使用字母块；
- 战术强化只展示两个图标化选项；宠物技能自动释放；
- 每局 **6–8 题**，目标时长 **5–8 分钟**。

### 4–6 年级

- 可在家长设置中启用温和倒计时；拼写可切键盘；
- 开放更多武器模块与弱点取舍；
- 每局 **8–12 题**，目标时长 **6–10 分钟**。

### 共同规则

- 口语设备失败不算学习错误；
- 所有动效可降低或关闭；
- 关键状态至少通过图形、文字、声音中的两种表达；
- 颜色不能作为唯一状态区分。

## 5. 核心循环

### 局外（七步）

进入基地 → 查看主线/到期突袭/三个短任务 → 配置武器与双宠 → 出击 → 获得掌握度/经验/合金/星晶 → 升级构筑 → 按新到期再次出发。

### 单局（八步）

入场 → 弱点侦测出题 → 全屏下令答题 → 指令结算 → 战斗演出 → 阶段检查（每 3 题可二选一强化）→ 终局 → 结算。

### 时长预算

- 主线：8–12 题，**6–10 分钟**；
- 到期复习：4–8 题，3–5 分钟；
- 弱项专项：6–10 题，5–8 分钟。

### 失败（紧急撤离）

保留知识进度与基础奖励；失去本局额外星章；不扣宠物、不损装备、不清航行日志、不强制现实等待。

## 6. 成长与内容规模

- 航员等级 **1–50**；每 **5** 级晋升军衔（称号/框/播报，无独立战力碾压）；
- 单元拆 **3–5** 个短任务，每任务最多 **3** 枚星章；
- 货币：**合金**（武器升级）、**星晶**（外观/玩具/弹道色/终结动画）；
- 每日三个短任务；七格航行日志点亮不清空；
- **5** 武器流派、**3** 首发宠物、Boss **三阶段**（护盾/装甲/核心）+ 知识装甲；
- 每 3 题二选一强化，同局最多 3 次。

## 7. 分期边界（P0–P5）

| 阶段 | 交付 |
| --- | --- |
| P0 | 文档真值对齐（本计划） |
| P1 | Cocos 工程地基、领域端口、六场景骨架 |
| P2 | `3A-U1` 纵向切片真机可玩 |
| P3 | 等级经济、武器、宠物、音画 |
| P4 | 全教材迁移 + 存档迁移 |
| P5 | 原生发包与真机矩阵 |

受众：**小学 1–6 年级**。
