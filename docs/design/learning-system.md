# 学习系统真值

> **Status:** Target  
> **Applies to:** Cocos 独立 App v2  
> **Last verified:** 2026-07-15

## 1. 知识点

稳定标识 `contentId`，必填：教材、年级、学期、单元、`kind`（词汇/句型/对话）、`en`、`zh`、`audioRef`、`questionTypes`、`distractorTags`。

学习状态另记：稳定度、难度、到期时间、首次答对次数、纠错次数、跳过次数、设备失败次数。

## 2. 掌握阶段（前端五阶段）

1. 新接触  
2. 初识  
3. 短期掌握  
4. 稳固  
5. 长期掌握  

内部调度使用离线可运行的 FSRS 式稳定度/难度模型；「怪兽威胁等级」只是表现层，不是第二套学习数据。

## 3. 选题优先级

1. 已到期复习  
2. 当前单元弱项  
3. 当前任务指定新内容  
4. 题型平衡项  

约束：同 `contentId` 不能连续；答错后延后 **2–3** 题位再现；一局内首次答对后的重复不再给完整经验；干扰项同类型同难度、无重复正确答案。

## 4. AnswerOutcome

| 标识 | 指令质量 | 学习影响 |
| --- | --- | --- |
| `first_correct` | 1.0 | 完整奖励；动量 +1 |
| `corrected` | 0.6 | 部分奖励；不增加动量 |
| `skipped` | 0.2 | 不判学习错误 |
| `device_failure` | 0.2 | 记设备失败，不增加错误次数 |
| `incorrect` | 0 | 错误 +1，降低稳定度并安排重现 |

## 5. 口语

- 主判定：目标文本与识别文本标准化相似度；
- 最低置信度以下不直接判 `incorrect`，允许一次重试；
- 识别不可用：跟读录音 + 自我确认，不计首次满额；
- 离线校验音频优先；系统 TTS 仅兜底；
- 口语质量只影响战斗演出强度。

## 6. contentId

全库唯一；迁移与掌握记录一律按 `contentId` 索引。
