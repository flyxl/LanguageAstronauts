# 成长与经济真值

> **Status:** Target  
> **Applies to:** Cocos 独立 App v2  
> **Last verified:** 2026-07-15

## 1. 航员等级

- 范围 1–50；累计经验不下降。
- 累计升级经验：`T(L) = round(60 × (L - 1)^1.55)`
- 实现：`calculateLevel(totalXp)` 取最大 L 使 `T(L) <= totalXp`。

XP 来源：

- 新知识首次完成：10；
- 到期复习完成：14；
- 提前复习：4；
- 拼写 ×1.25；口语 ×1.30；
- `first_correct` ×1.0；`corrected` ×0.65；`skipped`/`device_failure` ×0.2。

防刷：同一 `contentId` 同一天非到期练习从第 3 次起不再产生 XP。

功能解锁：2 级武器升级；3 级宠物舱；5 级局内战术芯片；8 级 Boss 悬赏。

## 2. 军衔

每 5 级晋升；只提供称号、头像框、基地装饰和入场播报；**不提供独立战斗数值**。

## 3. 星域

单元拆 3–5 短任务；每任务最多 3 星章（完成 / 首次正确率目标 / 对应到期复习）。下一任务只需完成前置，不强制满星。

## 4. 货币

- **合金**：主线与普通任务产出，用于武器升级。
- **星晶**：到期复习、首次满星、Boss 首杀产出，用于宠物外观、基地玩具、弹道颜色、终结动画。

星晶禁止购买武器数值、宠物孵化或进化。

武器升级成本：`weaponUpgradeCost(currentLevel) = 80 + 40 × (CurrentLevel - 1)` **合金**。

## 5. 日常

每天三任务：完成任意一场；清理到期复习；完成一个弱项题型。七格航行日志点亮不清空。
