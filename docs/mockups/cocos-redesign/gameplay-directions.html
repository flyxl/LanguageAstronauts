三条玩法路线

共同前提：全屏专注答题 → 作答结果转化为战斗资源 → 全屏战斗演出。

侦测　→　全屏下令　→　武器与宠物爆发

A · 电影化指令 RPG（最终骨架）

每题是一条舰队指令；武器负责攻击流派，宠物负责辅助与连携。

听力路线　│　拼写路线　│　口语精英

B · 星际远征 Roguelite

重复可玩性高，但随机奖励可能诱导刷简单题。

🚀　☄️　⚡　🐲

C · 轻动作弹幕

视觉冲击强，但操作能力会干扰学习结果。

最终组合

  • A 为核心骨架。
  • 每 3 题吸收一次 B 的二选一强化。
  • Boss 终结只加入一次简单 QTE。

明确避免

  • 答错即重罚。
  • 无限刷简单题。
  • 答题与复杂实时操作同时发生。
